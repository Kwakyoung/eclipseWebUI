Emmet 사용법

Elements(=요소)

Emmet에서 HTML 마크업 요소를 작성할때는, 요소명을 적고 Tab(=확장)을 누르면 자동으로 HTML요소로 만들어줍니다.

Nestin Operator(=중첩 연산자)

  • Child : > (자식, 자손 요소 => 포함관계로 작성)
  • Sibling : + (형제 => 중첩된 요소와 나란히 작성)
  • Climb-Up : ^ (캐럿 => 중첩을 벗어나는 것)
  • Multiplication : * (복제 => 여러 요소를 작성)
  • Grouping : () (그룹화)
